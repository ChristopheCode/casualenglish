# 🤝 Code of Conduct

---

## 📌 Our commitment

We want **Casual English** to be a welcoming and respectful project for everyone.

This project is focused on education, accessibility, and simple learning tools. Contributions should help keep the project clear, friendly, and useful.

---

## ✅ Expected behavior

Please try to:

- be respectful and constructive
- welcome different levels of experience
- give clear and helpful feedback
- focus discussions on the project and the learning experience
- keep Pull Requests and issues polite and easy to understand

---

## 🚫 Unacceptable behavior

Please avoid:

- insults or personal attacks
- harassment or intimidation
- disrespectful comments
- spam or unrelated promotion
- sharing private information without permission

---

## 🛠 Reporting a problem

If you notice behavior that does not follow this Code of Conduct, please contact the maintainer.

For security issues, please use the process described in `SECURITY.md`.

---

## 🌱 Project spirit

Casual English is a small educational project.

The goal is to make English learning easier, clearer, and more accessible for students.

Thank you for helping keep the project friendly and useful.
